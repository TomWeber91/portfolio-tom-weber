Projet Support Technique GLPI

Présentation du projet

Dans le cadre de notre formation, nous avons mis en place un site web de support technique pour le louvre. Ce site reprend les bases de GLPI pour faciliter la gestion des demandes de support et assurer une assistance technique rapide et efficace.

Configuration du site web

Le site web a été créé sur un serveur web dédié, avec une base de données MySQL pour stocker les informations de tickets et d'utilisateurs. Nous avons configuré les différents rôles (administrateurs, techniciens, utilisateurs) pour contrôler les accès et les permissions au sein du système.

Création des catégories de tickets

Pour faciliter la gestion des demandes, des catégories de tickets ont été créées (ex. : dépannage matériel, assistance logicielle, demandes d'améliorations). Cela permet de mieux organiser et prioriser les interventions.

Projet réalisé par les étudiants en 1ère année de BTS SIO - Option SISR
