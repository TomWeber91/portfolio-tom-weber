Projet Infrastructure Réseau Sation F

Présentation du Projet

Ce projet a pour objectif de mettre en place une infrastructure réseaux redondante pour Station F.

Switchs Cœur et Switchs Accès

Les switchs cœur ont été configurés pour gérer le routage entre les différents segments du réseau, assurant un flux de données optimisé et sécurisé.

Les switchs d'accès sont utilisés pour connecter les utilisateurs finaux.

Les switch coeurs sont stackés, les switchs sont configurés en etherchannel ainsi les liens sont redondants.

Le RSTP est activé sur les switchs afin d'éviter et d'anticiper la création de boucle.

Serveurs Active Directory (AD) / File Serveurs

Les serveurs Active Directory ont été redondés et configurés pour gérer les comptes utilisateurs et centraliser les droits d'accès sur le réseau.

Cela inclus la création et la gestion des unités organisationnelles et des stratégies de groupes (GPO) pour sécuriser l'environnement. Le File Serveur a été configuré pour la création des partages réseaux.

Serveurs HAProxy / Keepalived

HAProxy ont été configurés comme "load balancer" pour répartir les requêtes de manière équilibrée entre plusieurs serveurs web, améliorant ainsi la performance et la disponibilité de l'infrastructure réseau.

Projet réalisé par les étudiants en 2ème année de BTS SIO - Option SISR
